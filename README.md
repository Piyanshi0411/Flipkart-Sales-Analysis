# Flipkart Product Price Analysis

This data analytics project studies product prices, discounts, ratings, and category trends using a Flipkart-style e-commerce dataset.

The goal is to understand how product category, brand, discount percentage, and customer rating relate to product pricing on an online shopping platform.

## Project Objectives

- Analyze category-wise product pricing trends.
- Find products and categories with the highest discounts.
- Compare actual price and discounted price.
- Study the relationship between price, discount, and customer rating.
- Identify best value-for-money products.
- Create visualizations that can be used in a portfolio or presentation.

## Dataset

The sample dataset is available at:

```text
data/flipkart_products.csv
```

It contains the following columns:

- `product_name`
- `category`
- `brand`
- `actual_price`
- `discounted_price`
- `discount_percentage`
- `rating`
- `rating_count`
- `seller`
- `availability`

## How To Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/flipkart-product-price-analysis.git
cd flipkart-product-price-analysis
```

2. Install required libraries.

```bash
pip install -r requirements.txt
```

3. Run the analysis.

```bash
python src/flipkart_price_analysis.py
```

4. Check the generated charts and summary files inside the `outputs/` folder.

## Dashboard

This project also includes a browser-based dashboard:

```text
dashboard/index.html
```

To run it locally, start a simple server from the project folder:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/dashboard/
```

The dashboard includes:

- KPI cards for total products, average price, average discount, and average rating
- Category filter
- Average discount chart
- Actual vs discounted price comparison
- Top value products
- Product table sorted by value score

## Key Insights From Sample Data

- Electronics and appliances generally have higher average prices.
- Fashion and accessories often show higher discount percentages.
- High discount does not always guarantee high rating.
- Some products offer better value when rating, discount, and price are considered together.
